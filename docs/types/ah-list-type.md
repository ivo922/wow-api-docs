# Type: ah-list-type

← [API Types](../API Types.md) · [WoW API Docs](../index.md)

---

There are three different auction house listing types:

- `list` - The items that are currently for sale in the auction house.
- `bidder` - The items for which the player has bid.
- `owner` - The items that the player has placed up for auction.
