# Type: blizzid

← [API Types](../API Types.md) · [WoW API Docs](../index.md)

---

A unique numeric identifier, used by Blizzard in the game files and commonly found on database websites such as the WoW Armory and [wowhead.com](https://web.archive.org/web/20110319023203/http://wowhead.com/). These identifiers are not globally unique, however within a given class of ids, such as quest ids, they are used to uniquely identify a quest.
