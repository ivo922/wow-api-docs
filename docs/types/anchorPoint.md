# Type: anchorPoint

← [API Types](../API Types.md) · [WoW API Docs](../index.md)

---

String identifying a point relative to the dimensions of a Region; used in frame layout. Possible values:

- `CENTER`
- `BOTTOM`
- `TOP`
- `LEFT`
- `RIGHT`
- `BOTTOMLEFT`
- `BOTTOMRIGHT`
- `TOPLEFT`
- `TOPRIGHT`
