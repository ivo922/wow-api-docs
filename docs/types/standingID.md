# Type: standingID

← [API Types](../API Types.md) · [WoW API Docs](../index.md)

---

Identifies a level of reputation:

The default UI provides constants which can be helpful in displaying standing information.

- The localized name for the standing *N* can be found in the global variable `FACTION_STANDING_LABEL`*N* or `FACTION_STANDING_LABEL`*N*`_FEMALE`; e.g. `FACTION_STANDING_LABEL4 == "Neutral"`. Note: though the male (unlabeled) and female forms are the same in the `enUS` client, the same is not true for other languages -- be sure to use the appropriate form for the character's gender.
- Color values for each standing (as seen in reputation status bars in the default UI) can be found in the table `FACTION_BAR_COLORS`.
