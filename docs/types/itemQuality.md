# Type: itemQuality

← [API Types](../API Types.md) · [WoW API Docs](../index.md)

---

Indicates the quality (or rarity) of an item. Possible values and examples:

- `0`. Poor (gray): [Broken I.W.I.N. Button](https://web.archive.org/web/20110319023203/http://www.wowhead.com/?item=18230)
- `1`. Common (white): [Archmage Vargoth's Staff](https://web.archive.org/web/20110319023203/http://www.wowhead.com/?item=28455)
- `2`. Uncommon (green): [X-52 Rocket Helmet](https://web.archive.org/web/20110319023203/http://www.wowhead.com/?item=30847)
- `3`. Rare / Superior (blue): [Onyxia Scale Cloak](https://web.archive.org/web/20110319023203/http://www.wowhead.com/?item=15138)
- `4`. Epic (purple): [Talisman of Ephemeral Power](https://web.archive.org/web/20110319023203/http://www.wowhead.com/?item=18820)
- `5`. Legendary (orange): [Fragment of Val'anyr](https://web.archive.org/web/20110319023203/http://www.wowhead.com/?item=45038)
- `6`. Artifact (golden yellow): [The Twin Blades of Azzinoth](https://web.archive.org/web/20110319023203/http://www.wowhead.com/?item=18582)
- `7`. Heirloom (light yellow): [Bloodied Arcanite Reaper](https://web.archive.org/web/20110319023203/http://www.wowhead.com/?item=42943)
