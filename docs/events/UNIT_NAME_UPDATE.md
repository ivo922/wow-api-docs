# UNIT_NAME_UPDATE

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/UNIT_NAME_UPDATE)

---

Fires when a unit's name is changed. Also fires when the client is first notified of a unit's name.

**Signature:**

```lua
("unitID")
```

**Arguments:**

- `unitID` - The unit that was affected. (`string`)
