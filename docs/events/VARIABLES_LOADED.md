# VARIABLES_LOADED

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/VARIABLES_LOADED)

Fires when non-addon-specific saved variables are loaded. Addons should generally use [`ADDON_LOADED`](ADDON_LOADED.md) to determine whether their saved variables have loaded.
