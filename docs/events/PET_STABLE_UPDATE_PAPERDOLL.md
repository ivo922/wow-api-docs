# PET_STABLE_UPDATE_PAPERDOLL

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/PET_STABLE_UPDATE_PAPERDOLL)

---

Fires when information about 3D models used in the pet stables becomes available. Generally, this information is available on [`PET_STABLE_UPDATE`](PET_STABLE_UPDATE.md), but this event may fire if model information needed to be retrieved from the server.

()
