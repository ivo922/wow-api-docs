# ZONE_CHANGED

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/ZONE_CHANGED)

Fires when the player moves between subzones or other named areas. Only fired if [`ZONE_CHANGED_INDOORS`](ZONE_CHANGED_INDOORS.md) or [`ZONE_CHANGED_NEW_AREA`](ZONE_CHANGED_NEW_AREA.md) does not fire.
