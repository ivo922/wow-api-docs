# UPDATE_INVENTORY_ALERTS

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/UPDATE_INVENTORY_ALERTS)

---

Fires when an equipped item's durability alert status changes. Only fires for changes affecting durability alert status (conditions displayed as red or yellow on the default UI's durability warning frame); for other changes to item durability levels, see [`UPDATE_INVENTORY_DURABILITY`](UPDATE_INVENTORY_DURABILITY.md).

()
