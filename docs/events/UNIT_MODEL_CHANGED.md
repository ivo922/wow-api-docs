# UNIT_MODEL_CHANGED

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/UNIT_MODEL_CHANGED)

---

Fires when a unit's 3D model changes (e.g. due to shapeshifting, being polymorphed, or equipping gear)

**Signature:**

```lua
("unitID")
```

**Arguments:**

- `unitID` - The unit that was affected. (`string`)
