# QUEST_ITEM_UPDATE

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/QUEST_ITEM_UPDATE)

---

Fires when information about items in a questgiver dialog is updated. Only fires if information about quest items is retrieved from the server after the quest dialog is presented (via [`QUEST_DETAIL`](QUEST_DETAIL.md), [`QUEST_PROGRESS`](QUEST_PROGRESS.md), [`QUEST_COMPLETE`](QUEST_COMPLETE.md)).
