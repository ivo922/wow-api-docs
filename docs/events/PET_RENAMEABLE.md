# PET_RENAMEABLE

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/PET_RENAMEABLE)

---

Fires when the player is prompted to rename a pet which has been renamed before. A hunter's pet can normally only be named once, but a [Certificate of Ownership](http://www.wowhead.com/?item=43850) can be used to rename a pet which has been renamed before.
