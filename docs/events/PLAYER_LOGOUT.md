# PLAYER_LOGOUT

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/PLAYER_LOGOUT)

---

Fires immediately before the player is logged out of the game. Unlike [`PLAYER_LEAVING_WORLD`](PLAYER_LEAVING_WORLD.md), this event only fires upon logout and not when moving to different areas.
