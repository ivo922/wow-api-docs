# UNIT_FLAGS

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/UNIT_FLAGS)

Fires when certain combat statuses for a unit change (e.g. stunned, feared)

**Signature:**

```lua
("unit")
```

**Arguments:**

- `unit` - The id of the affected unit. (`string`)
