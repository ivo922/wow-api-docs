# UPDATE_GM_STATUS

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/UPDATE_GM_STATUS)

---

Fires when the player's GM ticket status (or ability to submit tickets) changes

**Signature:**

```lua
(avilable)
```

**Arguments:**

- `avilable` - Is 1 if the gms are available, 0 if they are not. (`number`)
