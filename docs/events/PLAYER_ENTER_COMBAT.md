# PLAYER_ENTER_COMBAT

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/PLAYER_ENTER_COMBAT)

---

Fires when the player begins melee auto-attack mode. This event cannot be used to detect when the player is entering a combat situation (i.e. when targeted by a hostile creature); for such cases, see [`PLAYER_REGEN_DISABLED`](PLAYER_REGEN_DISABLED.md).
