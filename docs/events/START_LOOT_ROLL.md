# START_LOOT_ROLL

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/START_LOOT_ROLL)

---

Fires when an item becomes available for group loot rolling

**Signature:**

```lua
(id, time)
```

**Arguments:**

- `id` - The id for this roll (`number`)
- `time` - How long the roll will last (`number`)
