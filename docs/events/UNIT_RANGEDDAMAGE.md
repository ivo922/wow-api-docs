# UNIT_RANGEDDAMAGE

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/UNIT_RANGEDDAMAGE)

---

Fires when a unit's ranged attack damage changes. Also fires when a unit's ranged attack speed changes.

**Signature:**

```lua
("unitID")
```

**Arguments:**

- `unitID` - The unit that was affected. (`string`)
