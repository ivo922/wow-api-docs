# UNIT_RESISTANCES

← [Events](../Events.md) · [Home](../index.md)

[Source](https://web.archive.org/web/20100726112636/http://wowprogramming.com/docs/events/UNIT_RESISTANCES)

---

Fires when a unit's magic resistances change. Also seems to fire when a unit's armor value changes.

**Signature:**

```lua
("unitID")
```

**Arguments:**

- `unitID` - The unit that was affected. (`string`)
